和自己的理解 支持向量机：svm方法中smo算法（最小序列化）算法每次对仅有两个点的最子集进行优化，每次选取两个a进行更新 不需要直接进行解二次规划问题通常情况
下，假设优化a1,a2,则其它a固定由Sum(a(i)*y(i)) = 0 可得y1*a1(new) + y2*a2(new) = y1*a1(old)+y2*a2(old)两边在同时乘以y1得：a1(new) + y1*y2*a2(new)
= a1(old) + y1*y2*a2(old) 令y1*y2 = s(s = 1 或者 -1 因为y表示类别）右边是一个定值用R表示则：a1(old) + s*a2(old) = a1(new) + s*a2(new)=R带入优化
目标函数：a2(new) = a2(old)+y2*(e2 -e1)/K 
当s=1: a2(new)的取值范围为  max(0,a2(old)-a1(old)) <= a2(new) <= min(c,c-a1(old)+a2(old)) 
当s=-1：a2(new)的取值范围为 max(0,a1(old)+a2(old)-c) <= a2(new) <= min(c,a1(old)+a2(old))
设a2(new)取值范围的最小值为L，最大值为H，通过上面两式可以得出L和H 则2（new)取值范围：

            L   a2(new) < L
a2（new) =  a1(new) + y2*(e2-e1)/k  L<=a2(new<=H
            H   a2(new) > H
